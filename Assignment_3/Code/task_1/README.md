# Wiring on the breadboard

1. GND pin >> blue negative rail on the breadboard.
2. LED. Long leg (anode) in hole j30 and the short leg (cathode) in hole j31.
3. 330 Ohm resistor. One leg to i31 other end to the blue negative rail.
4. Jumper wire. Hole i 30 to GPIO pin 13.
5. Button >> In the center gap at rows 20 and 22.
6. Jumper wire >> hole a20 to the blue negative rail.
7. Jumper wire >> hole j20 to GPIO pin 21.
