# LEDs

1. Green LED:
   Long leg >> J30
   Short leg >> J31
   Jumping Wire >> I30 to GPIO_13
   Resistor >> I31 to Negative/GND
2. Yellow LED:
   Long Leg >> J35
   Short Leg >> J36
   Jumping Wire >> I35 to GPIO_12
   Resistor >> I36 to Negative/GND
3. Red LED:
   Long Leg >> J40
   Short Leg >> J41
   Jumping Wire >> I40 to GPIO_14
   Resistor >> I41 to Negative/GND
