# MCP9808 Temperature Sensor

1. Jumper wire. 3V pin on Huzzah32 >> red positive rail.
2. VDD at c45 >> red positive rail.
3. GND at c46 >> blue negative rail.
4. SCL at c47 >> GPIO 33.
5. SDA at c48 >> GPIO 27.
6. A0, A1, A2 >> blue negative rail.
7. 1k Ohm resistor. One leg to b48 (SDA line) other end to red positive rail.
